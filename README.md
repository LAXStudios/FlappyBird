## Run

Öffne `CMD`

Klone das Projekt (Git muss installiert sein)

```bash
  git clone https://github.com/LAXStudios/FlappyBird
```

Navigiere zum Projekt

```bash
  cd FlappyBird
```

Starte das Spiel

```bash
  javac FlappyBird.java
```
oder
```bash
  java FlappyBird.java
```
